Detailed Description

Class for describing AST nodes. This is an abstract class and forms the root of the node hierarchy. The node hierarchy is organized in a structure according to the Composite design pattern: class node is the root of the hierarchy; class simple is a template for leaves holding simple (atomic) types; composite represents the recursive structure. Note that other recursion classes are possible by extending any of the classes in this family.
